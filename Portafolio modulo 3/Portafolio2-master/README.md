### De que trata el proyecto
***
Modulo 3 Fundamentos de desarrollo Back end JavaScript
Para crear un carrito de compras con JavaScript vanilla (sin utilizar librerías o frameworks externos), se pueden utilizar las siguientes herramientas y técnicas:

Módulos: Para modularizar el código y separar las distintas responsabilidades en distintos archivos o bloques de código. Esto hace que el código sea más fácil de entender y mantener.

Funciones: Para dividir el código en distintas partes reutilizables y fáciles de probar y depurar.

Eventos: Para escuchar y reaccionar a acciones del usuario, como hacer clic en un botón o cambiar el valor de un campo de texto.

DOM (Document Object Model): Para acceder y modificar elementos del HTML y mostrar información en la página web.

LocalStorage: Para almacenar información en el navegador del usuario y poder recuperarla más tarde, incluso después de que se cierre el navegador.

Funciones para agregar, quitar, actualizar la cantidad y el precio total de los productos en el carrito de compras. Esto puede incluir funciones para calcular el precio total de los productos en el carrito, actualizar la cantidad de productos en el carrito y mostrar la información del carrito al usuario.
